01. 스프링 MVC 기본기

01-1. 스프링 MVC 웹 요청 처리 과정

  1. 클라이언트로부터 요청을 받음
  1. Filter가 먼저 그 요청에 대해서 전후처리를 수행
  1. 2과정으로 처리된 요청이 dispatcher servlet을 들르게 됨
  1. handler mapping으로 요청에 적합한 컨트롤러를 검색(보통은 프론트컨트롤러 패턴으로 되어 있어서 , 매핑된 메서드를 찾는다고 생각하면 편하다)
  1. handler adapter에 의해서 처리요청을 컨트롤러로 전달하고자 함
  1. 인터셉터가 그 요청을 가로채서, 전후처리를 함
  1. 컨트롤러가 요청을 받아서 서비스단으로 작업을 요청
  1. 서비스단은 dao에게 데이터 작업을 요청
  1. dao(repository)는 dto를 이용해 db와 실질적인 작업을 수행하고 그 결과를 받아서 서비스단에게 전달
  1. 서비스단은 전달받은 결과를 컨트롤러에게 전달
  1. 컨트롤러는 알맞은 응답 결과 중 뷰에 대한 부분을 뷰리졸버에게 요청
  1. 뷰 리졸버는 전달받은 페이지를 알맞게 반환해줌

01-2. 마이바티스의 동작 구조

스프링과 마이바티스를 연결해서 사용할 때에는 org.mybatis.spring.SqlSessionFactoryBean과 org.mybatis.spring.SqlSessionTemplate을 이용하면 결국에는 SqlSession을 이용해서 mybatis를 간편하게 사용가능하다. 하지만, 그 구조를 정리해보도록 하자

🌟 마이바티스에서는 SqlSession을 만들어주어야 쿼리를 사용할 수 있다

  1. 먼저, 아래의 config.xml설정(별칭과 mapper 파일[쿼리, namespace 정보] 정보 정리) 과 같이 환경설정에 필요한 정보를 SessionFactoryBuilder가 참조하여 SessionFactory를 만든다
String resource = "org/mybatis/example/mybatis-config.xml";
InputStream inputStream = Resources.getResourceAsStream(resource);
SqlSessionFactory sqlSessionFactory =
  new SqlSessionFactoryBuilder().build(inputStream);

https://mybatis.org/mybatis-3/getting-started.html

  1. 그 다음에는, SessionFactory가 openSession()을 이용해서 SqlSession을 만든다
  1. 이 SqlSession을 이용해서 mapper파일에 있는 쿼리를 접근할 수 있는데 selectList, insert 등 다양한 메서드들을 활용할 수 있다

package com.js.test.dto;

import java.util.Date;


public class BoardDto {

	private String myContent;
	private Date myDate;

		super();

		this.myNo = myNo;
		this.myName = myName;
		this.myTitle = myTitle;
		this.myContent = myContent;
		this.myDate = myDate;
	}
	public int getMyNo() {
		return myNo;
	}
	public void setMyNo(int myNo) {
		this.myNo = myNo;
	}
	public String getMyName() {
		return myName;
	}
	public void setMyName(String myName) {
		this.myName = myName;
	}
	public String getMyTitle() {
		return myTitle;
	}
	public void setMyTitle(String myTitle) {
		this.myTitle = myTitle;
	}
	public String getMyContent() {
		return myContent;
	}
	public void setMyContent(String myContent) {
		this.myContent = myContent;
	}
	public Date getMyDate() {
		return myDate;
	}
	public void setMyDate(Date myDate) {
		this.myDate = myDate;
	}
	@Override
	public String toString() {
		return "BoardDto [myNo=" + myNo + ", myName=" + myName + ", myTitle=" + myTitle + ", myContent=" + myContent
				+ ", myDate=" + myDate + "]";
	}

	
	
}

02-7. dataSource를 기반으로 SqlSession을 가져오고, mapper&별칭를 설정한 정보(config.xml)를 담은 SqlSessionTemplate을 설정하는 root-context.xml 관리-applicationContext.xml

편의상, 기존에 "applicationContext.xml을 사용하여 빈 관리를 했던 터라, 이름을 applicationContext.xml로 바꿔주고, src/main/webapp/WEB-INF/spring /appServlet 위치로 위치를 변경해주기

  • db.properties 위치 저장←org.springframework.beans.factory.config.PropertyPlaceholderConfigurer

-locations setter로 리스트 설정

  • db.properties에서 연결을 위한 정보를 설정해줘서 dataSource 만들기 ←org.apache.commons.dbcp.BasicDataSource
  • dataSource와 mapper정보&별칭 설정이 담긴 config,xml(configLocation)을 이용해서 SqlSession 만들기 ←org.mybatis.spring.SqlSessionFactoryBean
  • SqlSession을 기반으로 SqlSessionTemplate 만들기←org.mybatis.spring.SqlSessionTemplate

package com.js.test;

import java.util.List;

import javax.servlet.http.HttpServletResponse;

import org.slf4j.Logger;
import org.slf4j.LoggerFactory;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Controller;
import org.springframework.ui.Model;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.RequestMethod;

import com.js.test.biz.BoardBiz;
import com.js.test.dto.BoardDto;

/**
 * Handles requests for the application home page.
 */
@Controller
public class HomeController {
	
	@Autowired
	private BoardBiz biz;
	
	private static final Logger logger = LoggerFactory.getLogger(HomeController.class);
	
	@RequestMapping(value="/list.do")
	public String selectList(Model model) {
		List<BoardDto> list=biz.selectList();
		logger.info("[controller]select list "+list);
		model.addAttribute("list", list);
		
		return "selectList";
	}
	
	
	@RequestMapping(value="/detail.do")
	public String selectOne(Model model, int myNo) {
		BoardDto dto=biz.selectOne(myNo);
		logger.info("[controller]select one "+dto);
		model.addAttribute("target",dto);
		
		return "target";
	}
	
	@RequestMapping(value="/insertForm.do")
	public String insertForm() {
		logger.info("[controller]insertForm");
		return "insertForm";
	}
	
	@RequestMapping(value="/insert.do",method=RequestMethod.POST)
	public String insert(HttpServletResponse response, String myName,String myTitle,String myContent) {
		BoardDto dto=new BoardDto();
		
		logger.info("[controller]insert");
		
		dto.setMyName(myName);
		dto.setMyTitle(myTitle);
		dto.setMyContent(myContent);
		int resIns=biz.insert(dto);
		
		if(resIns>0) {
			return "redirect:list.do";
		}else {
			return "redirect:insertForm.do";
		}
	}
	
	@RequestMapping(value="/updateForm")
	public String updateForm(Model model,int myNo) {
		BoardDto dto=biz.selectOne(myNo);
		
		logger.info("[controller]updateForm-"+dto);
		
		model.addAttribute("update", dto);
		
		return "updateForm";
	}
	
	@RequestMapping(value="/update.do")
	public String update(BoardDto dto) {
		int resUp=biz.update(dto);
		int myNo=dto.getMyNo();
		
		logger.info("[controller]update-"+dto);
		
		
		if(resUp>0) {
			return "redirect:detail.do?myNo="+myNo;
		}else {
			return "redirect:updateForm.do?myNo="+myNo;
		}
	}
	
	
	@RequestMapping(value="/delete.do")
	public String delete(int myNo) {
		int resDel=biz.delete(myNo);
		
		if(resDel>0) {
			return "redirect:list.do";
		}else {
			return "redirect:detail.do?myNo="+myNo;
		}
	}
	
}
