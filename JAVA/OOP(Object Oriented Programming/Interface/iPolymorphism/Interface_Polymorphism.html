인터페이스의 다형성 특징

  1. Situation

    class Child1 implements Parent

    {

    ...

    }

    class Child2 implements Parent

    {

    ...

    }

    1. Reference Variable & Instance

      [Interface]Parent p=new [Class]Child1();

    2. Return Type

      public Parent method()

      {

      ...

      return new Child2();

      /**

      * Or

      * Parent p=new Child2();

      * return p;

      */

      }

    3. arguments(parameter)-파라미터로 해당 인터페이스가 구현된 클래스 인스턴스를 전달

      public class Child3 implements Parent

      {

      void attack(Parent p)

      {

      ...

      }

      public static void main(String[] args)

      {

      Child3 c1=new Child3();

      c1.attack(new Child3());

      }
